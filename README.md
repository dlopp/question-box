## :hammer_and_wrench: Setup
```bash
  # リポジトリをローカルにクローン
    git clone https://github.com/dlopp/question-box.git
  # クローンしたリポジトリに移動
    cd question-box
  # npm経由でyarnをインストール
    npm install -g yarn
  # 依存関係をインストール
    yarn install
  # ローカルサーバーを起動しアクセス
    yarn dev --open
  # ctrl + c でサーバーを停止できます。
```
